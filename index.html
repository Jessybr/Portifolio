Jogo da velha

Jogo da velha

O site apresenta um jogo da velha desenvolvido utilizando as tecnologias ReactJS e Tailwind CSS. O jogo é acessível tanto em dispositivos móveis quanto em computadores, pois é responsivo.

  • Alerta de vencedor e empate:
    • Quando um jogador ganha, um alerta é exibido informando sobre a vitória, e da mesma forma, quando ocorre um empate, um alerta é mostrado indicando o resultado. No final de cada jogo o botão para reiniciar o jogo é exibido.

Essa abordagem proporciona uma experiência interativa e amigável aos usuários, permitindo que eles desfrutem do clássico jogo da velha em diferentes dispositivos e telas.

Expert Notes

Expert Notes

O site Expert Notes pode guardar anotações feitas pelo usuário. Foi desenvolvido com ReactJS, com tipagem Typescript, tooling com Vite, TailwindCSS e APIs da Web como Web Storage e Web Speech.

  • Anotação por texto e por voz:
    • É possível fazer anotações tanto por texto quanto por voz. É utilizado a API Web Speech para tranformar gravações de voz em texto de forma eficiente.
  • Armazenamento de notas:
    • As notas feitas são armazenadas localmente utilizando a API Web Storage, garantindo assim que nenhuma anotação seja perdida quando o navegador é fechado.
  • Mecanismo de pesquisa de notas:
    • É possível pesquisar anotações específicas a partir das palavras digitadas na barra de pesquisa.

Amor de Casquinha

Amor de Casquinha

Amor de Casquinha é uma sorveteria fictícia desenvolvida pelo grupo composto por mim, Gabriel Barbosa, Laura Barros e Caroliny Sampaio.

O site em questão é um sistema de comércio eletrônico baseado em PHP e banco de dados. Ele possui funcionalidades específicas para funcionários e clientes, proporcionando uma experiência de compra e gerenciamento eficiente. Algumas características principais incluem:

  • Acessibilidade e Design Responsivo:
    • O site foi projetado para oferecer uma experiência acessível em telas grandes e pequenas.O design prioriza a legibilidade, garantindo uma navegação fácil e agradável.
  • Autenticação Segura:
    • O site oferece autenticação segura para funcionários e clientes.Os clientes podem criar uma conta e fazer login para acessar recursos personalizados.
  • Carrinho de Compras Persistente:
    • Mesmo que um cliente não logado adicione produtos ao carrinho de compras, esses itens permanecerão quando o cliente fizer login, facilitando a conclusão da compra.
  • Gestão de Pedidos:
    • Após a conclusão da compra, é gerada uma "nota fiscal" contendo detalhes dos pedidos, dados do cliente e o valor total.Os funcionários têm acesso aos pedidos feitos por clientes para gerenciar eficientemente as transações.
  • Gerenciamento de Produtos:
    • Funcionários comuns têm a capacidade de criar, editar e excluir produtos, incluindo suas variações.Isso permite manter atualizado o catálogo de produtos disponíveis para os clientes.
  • Privilegiados Administrativos:
    • Funcionários administradores têm privilégios estendidos, permitindo que gerenciem funcionários comuns.Eles podem criar, editar e excluir perfis de funcionários, mantendo o controle sobre quem tem acesso ao sistema.

Essas funcionalidades combinadas oferecem uma solução abrangente para a gestão de vendas online, permitindo que clientes realizem compras de maneira eficaz, enquanto os funcionários têm ferramentas poderosas para gerenciar produtos, pedidos e usuários.

Jogo da Memória

Jogo da Memória

O site é um jogo da memória dedicado ao meio ambiente, incentivando a conscientização e conhecimento sobre temas ecológicos

A linguagem utilizada para fazer a lógica foi PHP.

  • Níveis de Dificuldade:
    • Oferece dois níveis para os jogadores: fácil e difícil, proporcionando opções para diferentes habilidades e desafios.
  • Número de Cartas e Pares:
    • O nível fácil possui 8 cartas, totalizando 4 pares corretos. O nível difícil apresenta 12 cartas, com 6 pares corretos, aumentando a complexidade do jogo.
  • Sistema de Pontuação:
    • O jogador inicia com 10 pontos. Ao virar duas cartas que não formam um par, o jogador perde 2 pontos. Se encontrar um par, ganha 5 pontos. O objetivo é encontrar todos os pares antes de zerar os pontos.
  • Objetivo do Jogo:
    • O jogador vence ao encontrar todos os pares disponíveis. O jogo termina se o jogador zerar os pontos, resultando em derrota.

Essa abordagem educativa e interativa oferece uma experiência de jogo envolvente, ao mesmo tempo em que promove a consciência ambiental e o desenvolvimento da memória.

Casa Verde

O site é uma plataforma abrangente que fornece informações detalhadas sobre uma ampla variedade de plantas, abordando desde o plantio inicial até a manutenção contínua. Algumas características notáveis incluem:

    • Esse site se destaca por sua abordagem inclusiva, tornando o conhecimento sobre plantas acessível a todos, independentemente do tamanho da tela ou das necessidades específicas de leitura.
